# React Task Manager App

A modern frontend application built with React + TypeScript, featuring authentication, protected routes, GraphQL integration, real-time capabilities, and a polished UI powered by shadcn/ui.
The project follows best practices and includes unit, integration, and snapshot tests.

## Tech Stack
- Frontend: React, TypeScript
- Routing: @tanstack/react-router
- Forms & Validation: React Hook Form, Zod
- UI: shadcn/ui, Tailwind CSS
- State & Data: Apollo Client (GraphQL)
- Testing: Jest, React Testing Library
- API: GraphQL (mock & external endpoints)
- Other: WebSocket-ready architecture

## Getting Started
### 1. Install dependencies
```
npm install
```
### 2. Run development server
```
npm run dev
```
### 3. Run tests
```
npm test
```
